## Getting Started

### Development

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

### Production Build

```bash
npm run build
```

The optimized production build will be in the `dist/` directory.

## Usage

1. **Create a New Document** - Start with a blank paper and customize as needed
2. **Load a Sample** - View an example formatted document
3. **Import Content** - Load from HTML, PDF, or Markdown files
4. **Edit & Format** - Use the built-in editor to structure your paper
5. **Export to PDF** - Generate a professional PDF ready for submission or printing

## Document Structure

Documents are organized with:
- **Title & Subtitle** - Main heading and optional subtitle
- **Metadata** - Author, date, version, classification, etc.
- **Sections** - Hierarchical content (h1, h2, h3 levels)
- **Paragraphs** - Multiple paragraphs per section
- **Code Blocks** - Syntax-highlighted code examples
